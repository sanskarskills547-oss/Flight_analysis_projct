# Airline Passenger Traffic Analysis

## Project Objective

The goal of this project is to analyze airline passenger traffic using SQL.

The project starts with a raw metadata table and transforms it into a normalized relational database structure.

## Database Design

Tables Created:

* Airline
* Airport
* Flight
* FlightMetrics
* City

Relationships were established using primary and foreign keys.

## Key Business Questions

1. Which routes carry the highest passenger volumes?
2. Which airports generate the most traffic?
3. What are the busiest flight corridors?
4. How does passenger volume vary by month?
5. Is there a relationship between city population and air traffic?

## Technologies Used

* MySQL
* SQL Joins
* Aggregations
* Group By
* Foreign Keys
* Data Modeling
* ETL Concepts

## Key Learnings

* Relational database design
* Primary and foreign key relationships
* Data normalization
* Analytical SQL queries
* Business-oriented reporting
