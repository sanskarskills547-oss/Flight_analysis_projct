# Airline Passenger Traffic Analysis (SQL)

## Overview

This project analyzes airline traffic data using SQL.

The dataset was transformed from a single metadata table into a normalized relational database consisting of airline, airport, flight, flight metrics, and city tables.

The analysis focuses on:

* Passenger traffic trends
* High-volume routes
* Airport performance
* Flight frequency analysis
* Population vs Air Traffic analysis

## Database Schema

Tables:

* Airline
* Airport
* Flight
* FlightMetrics
* City

## Sample Analysis

### Top Routes by Passenger Volume

Identifies routes carrying the highest number of passengers.

### Airport Performance

Calculates total flights and average passengers per flight.

### Flight Frequency Analysis

Determines the busiest airport corridors.

### Population vs Air Traffic

Explores the relationship between city population and passenger traffic.
